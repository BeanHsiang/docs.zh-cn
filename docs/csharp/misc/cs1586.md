# <a name="compiler-error-cs1586"></a>编译器错误 CS1586
数组创建必须有数组大小或数组初始值设定项  
  
 未正确声明数组。  
  
 下面的示例生成 CS1586：  
  
```csharp  
// CS1586.cs  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      int[] a = new int[];   // CS1586  
      // try the following line instead  
      int[] b = new int[5];  
   }  
}  
```
