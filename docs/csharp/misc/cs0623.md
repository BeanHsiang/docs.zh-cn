# <a name="compiler-error-cs0623"></a>编译器错误 CS0623
数组初始值设定项只能在变量或字段初始值设定项中使用。 请尝试改用 new 表达式。  
  
 尝试在某个上下文中使用数组初始值设定项来初始化数组，而该上下文中不允许这样做。  
  
## <a name="example"></a>示例  
 下面的示例会生成 CS0623，因为编译器会{4}为外部数组初始值设定项内的嵌入式的数组初始值设定项：  
  
```csharp  
//cs0632.cs  
using System;  
  
class X  
{  
    public int[] x = { 2, 3, {4}}; //CS0623  
}  
```
