# <a name="compiler-error-cs1002"></a>编译器错误 CS1002
应输入 ;  
  
 编译器检测到缺少一个分号。 C# 中每条语句的末尾都需要一个分号。 语句可以跨多行。  
  
 下面的示例生成 CS1002：  
  
```csharp  
// CS1002.cs  
namespace x  
{  
   abstract public class clx  
   {  
      int i   // CS1002, missing semicolon  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
