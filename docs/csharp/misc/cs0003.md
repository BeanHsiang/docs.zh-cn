# <a name="compiler-error-cs0003"></a>编译器错误 CS0003
内存不足  
  
 编译器分配的虚拟内存不足，无法完成编译。 关闭所有不必要的应用程序并重新编译。  
  
 你可能想要增加页面文件大小，请确保具有可用的磁盘空间。  
  
 [!INCLUDE[winsdklong](~/includes/winsdklong-md.md)] 和 C# 编译器的版本不匹配或支持 C# 编译器的一个或多个文件损坏，也会出现此错误；请重新安装 Visual Studio。
