# <a name="compiler-error-cs0249"></a>编译器错误 CS0249
请勿重写 object.Finalize。 相反，请提供析构函数。  
  
 用析构函数语法来指定对象被破坏时要执行的指令。  

 以下示例生成 CS0249：  
  
```csharp  
// CS0249.cs  
class MyClass  
{  
   protected override void Finalize()   // CS0249  
   // try the following line instead  
   // ~MyClass()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
