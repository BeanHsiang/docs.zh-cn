# <a name="corprfgcreason-enumeration"></a>COR_PRF_GC_REASON 枚举
指示当前发生垃圾回收的原因。  
  
## <a name="syntax"></a>语法  
  
```  
typedef enum {  
    COR_PRF_GC_INDUCED = 1,  
    COR_PRF_GC_OTHER = 0  
} COR_PRF_GC_REASON;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|`COR_PRF_GC_INDUCED`|通过已引发垃圾回收<xref:System.GC.Collect%2A>方法。|  
|`COR_PRF_GC_OTHER`|原因是未指定。|
