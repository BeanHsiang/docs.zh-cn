# <a name="icorprofilercallback2threadnamechanged-method"></a>ICorProfilerCallback2::ThreadNameChanged 方法
通知代码探查器线程的名称已更改。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT ThreadNameChanged(  
    [in] ThreadID threadId,  
    [in] ULONG cchName,  
    [in] WCHAR name[]);  
```  
  
## <a name="parameters"></a>参数  
 `threadId`  
 [in]线程的 ID。  
  
 `cchName`  
 [in]该线程的新名称的长度。  
  
 `name`  
 [in]新线程的名称。 名称不以 null 结尾。
