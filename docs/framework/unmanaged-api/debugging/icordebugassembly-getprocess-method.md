# <a name="icordebugassemblygetprocess-method"></a>ICorDebugAssembly::GetProcess 方法
获取在其中运行此 icor 调试程序集实例的进程的接口指针。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetProcess (  
    [out] ICorDebugProcess **ppProcess  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppProcess`  
 [out]指向表示流程 ICorDebugProcess 接口的指针。
