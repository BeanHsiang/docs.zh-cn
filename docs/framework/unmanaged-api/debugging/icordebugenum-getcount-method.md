# <a name="icordebugenumgetcount-method"></a>ICorDebugEnum::GetCount 方法
枚举中获取项的数目。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetCount (  
    [out] ULONG *pcelt  
);  
```  
  
## <a name="parameters"></a>参数  
 `pcelt`  
 [out]一个指向枚举中的项的数目。
