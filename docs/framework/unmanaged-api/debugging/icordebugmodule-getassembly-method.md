# <a name="icordebugmodulegetassembly-method"></a>ICorDebugModule::GetAssembly 方法
获取包含此模块的程序集。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetAssembly(  
    [out] ICorDebugAssembly **ppAssembly  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppAssembly`  
 [out]指向一个 icor 调试程序集对象，表示包含此模块的程序集的指针。
