# <a name="icordebugmodulegetprocess-method"></a>ICorDebugModule::GetProcess 方法
获取包含此模块的过程。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetProcess (  
    [out] ICorDebugProcess **ppProcess  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppProcess`  
 [out]指向表示包含此模块的流程 ICorDebugProcess 对象的地址的指针。
