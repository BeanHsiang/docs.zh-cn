# <a name="icordebugbreakpointactivate-method"></a>ICorDebugBreakpoint::Activate 方法
设置此活动的状态`ICorDebugBreakpoint`。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Activate (  
    [in] BOOL bActive  
);  
```  
  
## <a name="parameters"></a>参数  
 `bActive`  
 [in]将此值设置为`true`来指定的状态为活动状态; 否则，将此值设置为`false`。
