# <a name="icordebugevalnewstring-method"></a>ICorDebugEval::NewString 方法
分配新的字符串实例具有指定的内容。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT NewString (  
    [in] LPCWSTR   string  
);  
```  
  
## <a name="parameters"></a>参数  
 `string`  
 [in]指向字符串的内容。  
  
## <a name="remarks"></a>备注  
 始终在线程当前执行的应用程序域中创建的字符串。
