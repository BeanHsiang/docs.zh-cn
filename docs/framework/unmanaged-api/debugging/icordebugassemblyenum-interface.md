# <a name="icordebugassemblyenum-interface"></a>ICorDebugAssemblyEnum 接口

实现 ICorDebugEnum 方法，并枚举 icor 调试程序集数组。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugassemblyenum-next-method.md)|获取指定的数目的`ICorDebugAssembly`实例在枚举中，从当前的位置开始。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
>  此接口不支持跨计算机或跨进程远程调用。
