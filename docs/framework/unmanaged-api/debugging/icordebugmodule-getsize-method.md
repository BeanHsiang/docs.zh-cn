# <a name="icordebugmodulegetsize-method"></a>ICorDebugModule::GetSize 方法
获取用字节表示，该模块的大小。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetSize(  
    [out] ULONG32 *pcBytes  
);  
```  
  
## <a name="parameters"></a>参数  
 `pcBytes`  
 [out]以字节为单位的模块的大小。  
  
 如果该模块通过本机映像生成器 (NGen.exe) 生成的该模块的大小将为零。
