# <a name="icordebugbreakpointisactive-method"></a>ICorDebugBreakpoint::IsActive 方法
获取一个值，该值指示是否此`ICorDebugBreakpoint`处于活动状态。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT IsActive (  
    [out] BOOL *pbActive  
);  
```  
  
## <a name="parameters"></a>参数  
 `pbActive`  
 [out]`true`如果此断点是活动; 否则为`false`。
