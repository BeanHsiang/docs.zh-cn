# <a name="imetadataemitsave-method"></a>IMetaDataEmit::Save 方法
将所有元数据保存到指定地址处的文件在当前范围内。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Save (   
    [in]  LPCWSTR     szFile,   
    [in]  DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>参数  
 `wzFile`  
 [in]若要将保存到文件的名称。 如果此值为 null，内存中副本将保存到上次使用的位置。  
  
 `dwSaveFlags`  
 [in] 保留。 必须为零。
