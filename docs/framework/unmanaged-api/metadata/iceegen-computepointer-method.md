# <a name="iceegencomputepointer-method"></a>ICeeGen::ComputePointer 方法
确定指定的代码部分的缓冲区。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT ComputePointer (  
    [in]  HCEESECTION  section,  
    [in]  ULONG        RVA,   
    [out] UCHAR        **lpBuffer  
);  
```  
  
## <a name="parameters"></a>参数  
 `section`  
 [in]要为其返回缓冲区在代码部分。  
  
 `RVA`  
 [in]要为其获取一个指针，该方法相对虚拟地址。  
  
 `lpBuffer`  
 [out]指向返回的缓冲区的指针。
