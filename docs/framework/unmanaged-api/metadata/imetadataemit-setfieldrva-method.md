# <a name="imetadataemitsetfieldrva-method"></a>IMetaDataEmit::SetFieldRVA 方法
设置指定标记所引用的字段的相对虚拟地址的全局变量值。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT SetFieldRVA (   
    [in]  mdFieldDef  fd,   
    [in]  ULONG       ulRVA   
);  
```  
  
## <a name="parameters"></a>参数  
 `fd`  
 [in]目标字段的标记。  
  
 `ulRVA`  
 [in]代码或数据区域的地址。
