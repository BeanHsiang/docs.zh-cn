# <a name="imetadataemit2resetenclog-method"></a>IMetaDataEmit2::ResetENCLog 方法
重置编辑并继续日志并启动新会话。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT ResetENCLog ();  
```  
  
## <a name="parameters"></a>参数  
 `ResetENCLog` 不带任何参数。
