# <a name="cvstruct-structure"></a>CVStruct 结构
包含在安装模块或复合图像时所使用的信息。  
  
## <a name="syntax"></a>语法  
  
```  
typedef struct {  
    short Major;  
    short Minor;  
    short Sub;  
    short Build;  
} CVStruct;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|主要|主要版本的内部版本号。|  
|次要|次要版本生成号。|  
|Sub|二次内部版本号。|  
|生成|生成号。|
