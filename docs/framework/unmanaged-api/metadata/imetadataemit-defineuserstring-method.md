# <a name="imetadataemitdefineuserstring-method"></a>IMetaDataEmit::DefineUserString 方法
获取指定文本字符串的元数据令牌。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT DefineUserString (   
    [in]  LPCWSTR     szString,   
    [in]  ULONG       cchString,   
    [out] mdString    *pstk   
);  
```  
  
## <a name="parameters"></a>参数  
 `szString`  
 [in]要存储的用户字符串。  
  
 `cchString`  
 [in]中的宽字符计数`szString`。  
  
 `pstk`  
 [out]分配的字符串标记。
