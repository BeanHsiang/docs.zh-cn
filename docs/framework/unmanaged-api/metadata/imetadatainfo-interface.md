# <a name="imetadatainfo-interface"></a>IMetaDataInfo 接口
提供一种方法，使用此方法可获取关于将元数据从磁盘文件映射到内存中的信息。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetFileMapping 方法](../../../../docs/framework/unmanaged-api/metadata/imetadatainfo-getfilemapping-method.md)|获取内存区域的映射的文件和映射的类型。|
