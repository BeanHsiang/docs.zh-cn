# <a name="coiniticor-enumeration"></a>COINITICOR 枚举
指定使用的常量[CoInitializeCor](../../../../docs/framework/unmanaged-api/hosting/coinitializecor-function.md)当它初始化公共语言运行时。  
  
## <a name="syntax"></a>语法  
  
```  
typedef enum tagCOINITCOR  
{  
    COINITCOR = 0x0  
} COINITICOR;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|`COINITCOR`|指示默认初始化模式。|
