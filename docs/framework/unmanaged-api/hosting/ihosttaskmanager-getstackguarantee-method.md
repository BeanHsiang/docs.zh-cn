# <a name="ihosttaskmanagergetstackguarantee-method"></a>IHostTaskManager::GetStackGuarantee 方法
获取的堆栈空间后的堆栈操作完成后，可保证，但在关闭进程之前的量。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetStackGuarantee(  
    [out] ULONG *pGuarantee  
);  
```  
  
## <a name="parameters"></a>参数  
 `pGuarantee`  
 [out]指向可用的字节数的指针。
