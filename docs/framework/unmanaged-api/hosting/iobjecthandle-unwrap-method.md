# <a name="iobjecthandleunwrap-method"></a>IObjectHandle::Unwrap 方法
解包从间接寻址的按值封送的对象。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Unwrap (  
    [out, retval] VARIANT *ppv  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppv`  
 [out]指向要解包的对象的指针。
