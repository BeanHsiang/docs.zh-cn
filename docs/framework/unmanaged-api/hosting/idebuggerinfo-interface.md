# <a name="idebuggerinfo-interface"></a>IDebuggerInfo 接口
提供用于获取有关调试的服务的状态信息的方法。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[IsDebuggerAttached 方法](../../../../docs/framework/unmanaged-api/hosting/idebuggerinfo-isdebuggerattached-method.md)|获取一个值，该值指示是否有托管的调试器附加到此进程。|
