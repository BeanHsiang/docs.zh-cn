# <a name="iassemblynamefinalize-method"></a>IAssemblyName::Finalize 方法
允许此操作[IAssemblyName](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md)对象释放资源并调用其析构函数之前执行其他清理操作。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Finalize ();  
```
