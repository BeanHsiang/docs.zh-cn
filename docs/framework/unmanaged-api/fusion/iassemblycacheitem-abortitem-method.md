# <a name="iassemblycacheitemabortitem-method"></a>IAssemblyCacheItem::AbortItem 方法
允许在全局程序集缓存中的程序集以进行发布之前执行清理操作。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT AbortItem ();  
```
