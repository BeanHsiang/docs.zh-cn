# <a name="isymunmanagedwriter3commit-method"></a>ISymUnmanagedWriter3::Commit 方法
提交所做的更改写入到流的到目前为止。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Commit();  
```  
  
## <a name="return-value"></a>返回值  
 如果方法成功，则为 S_OK否则为 E_FAIL 或某些其他错误代码。  
  
## <a name="requirements"></a>要求  
 **标头：** CorSym.idl CorSym.h
