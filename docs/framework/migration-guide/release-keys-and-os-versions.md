 |.NET Framework 版本|Release DWORD 的值|
 |--------------------------------|-------------|
|.NET Framework 4.5|所有 Windows 操作系统：378389|
|.NET Framework 4.5.1|在 Windows 8.1 和 Windows Server 2012 R2 上：378675<br />在所有其他 Windows 操作系统上：378758|
|.NET Framework 4.5.2|所有 Windows 操作系统：379893|
|.NET Framework 4.6|在 Windows 10 上：393295<br />在所有其他 Windows 操作系统上：393297|
|.NET Framework 4.6.1|在 Windows 10 11 月更新系统上：394254<br />在所有其他 Windows 操作系统上：394271|
|.NET Framework 4.6.2|在 Windows 10 周年更新和 Windows Server 2016 上：394802<br />在所有其他 Windows 操作系统上：394806|
|.NET Framework 4.7|在 Windows 10 创意者更新上：460798<br />在所有其他 Windows 操作系统上：460805| 
|.NET Framework 4.7.1|在 Windows 10 Fall Creators Update 和 Windows Server 版本 1709 上：461308<br/>在所有其他 Windows 操作系统上：461310|
|.NET Framework 4.7.2|在 Windows 10 2018 年 4 月更新和 Windows Server 版本 1803 上：461808<br/>在所有其他 Windows 操作系统上：461814|
