# <a name="splitter-control-overview-windows-forms"></a>Splitter 控件概述（Windows 窗体）
> [!IMPORTANT]
>  尽管<xref:System.Windows.Forms.SplitContainer>替换并添加了功能<xref:System.Windows.Forms.Splitter>的早期版本中，控制<xref:System.Windows.Forms.Splitter>如果选择保留向后兼容性和将来使用。  
  
 Windows 窗体<xref:System.Windows.Forms.Splitter>控制用于在运行时调整停靠的控件的大小。 <xref:System.Windows.Forms.Splitter>通常具有的数据来展示，如 Windows 资源管理器，它的数据窗格包含不同的宽度的信息在不同的时间长度可变的控件与窗体上使用控件。  
  
## <a name="working-with-the-splitter-control"></a>使用拆分器控件  
 当用户将鼠标指针指向未停靠的控件可调整大小的拆分器控件的边缘时，指针将更改其外观，以指示该控件可以调整大小。 拆分器控件后，用户可以调整大小的停靠的控件的紧前面。 因此，若要允许用户调整停靠的控件的大小在运行时，停靠控件要调整大小的容器的边缘，然后将拆分器控件停靠到该容器在同一端。
