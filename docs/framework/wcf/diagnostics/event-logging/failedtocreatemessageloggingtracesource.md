# <a name="failedtocreatemessageloggingtracesource"></a>FailedToCreateMessageLoggingTraceSource
ID:115  
  
 严重性：Error  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示消息日志记录无法创建跟踪源。 此事件列出了异常、进程名称和进程 ID。
