# <a name="failedtoinitializetracesource"></a>FailedToInitializeTraceSource
ID:101  
  
 严重性：Error  
  
 类别：跟踪  
  
## <a name="description"></a>描述  
 未初始化跟踪源。 跟踪功能被禁用。 此事件列出了异常、进程名称和进程 ID。
