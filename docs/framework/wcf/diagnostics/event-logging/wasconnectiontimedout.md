# <a name="wasconnectiontimedout"></a>WasConnectionTimedout
ID:163  
  
 严重性：Error  
  
 类别：SMSvcHost  
  
## <a name="description"></a>描述  
 该事件指示 Windows 进程激活服务 (WAS) 需要超时的一个连接，并列出进程名称和 ID。
