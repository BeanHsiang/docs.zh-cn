# <a name="failedtosetuptracing"></a>FailedToSetupTracing
ID:100  
  
 严重性：Error  
  
 类别：跟踪  
  
## <a name="description"></a>描述  
 设置跟踪的尝试已失败。 跟踪功能被禁用。  
  
 此事件列出了异常、进程名称和进程 ID。
