# <a name="failedtotraceevent"></a>FailedToTraceEvent
ID:104  
  
 严重性：Error  
  
 类别：跟踪  
  
## <a name="description"></a>描述  
 未跟踪一个或多个事件。 此事件将列出原始事件字符串、进程名称和进程 ID。
