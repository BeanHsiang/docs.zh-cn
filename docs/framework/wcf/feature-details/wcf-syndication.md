# <a name="wcf-syndication"></a>WCF 联合
Windows Communication Foundation (WCF) 提供了支持用于轻松处理 Atom、 RSS 或其他自定义格式的联合源可用于读取和创建它们，以及将它们公开的服务终结点。 本节中的主题详细描述用于联合的此种编程模型。
