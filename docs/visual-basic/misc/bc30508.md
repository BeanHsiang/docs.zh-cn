# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<名称 > 不能公开类型\<type1 > 中\<说明符 > \<y p e 2 >\<类型名称 > 通过\<名称 >
试图公开公共类 `MyClass`外部的私有类型。  
  
 **错误 ID:** BC30508  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将该类型声明为 `Public`。
