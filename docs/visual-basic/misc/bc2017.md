# <a name="could-not-find-library-libraryname"></a>找不到库\<libraryname >
未能找到指定的库。  
  
 当从命令提示符下进行编译，并使用引用多个程序集的响应 (.rsp) 文件时，可能会发生此问题  
  
 **错误 ID:** BC2017  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查库名称以确保其正确无误。
