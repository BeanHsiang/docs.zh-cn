# <a name="argument-not-specified-for-parameter-parametername-of-methodname"></a>没有为参数指定的参数\<parametername >' 的 '\<方法名称 >
没有为所需形参提供实参。  
  
 **错误 ID:** BC30455  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   为指定形参提供实参。
