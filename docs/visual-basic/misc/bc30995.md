# <a name="leading--or--cannot-appear-in-a-constant-expression"></a>在常量表达式中不能出现前导“.”或“!”
成员访问 （.） 和字典成员访问 （！） 需要一个表达式，指定包含大多数情况下，其中包括常量表达式的成员的元素。 以下声明无效。  
  
```  
' Not valid.  
Const c As String = .name  
```  
  
 **错误 ID:** BC30995  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   指定包含想要访问的成员的实例。
