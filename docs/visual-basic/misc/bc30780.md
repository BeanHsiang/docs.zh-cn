# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>“Continue”语句在即时窗口中无效
Continue 语句仅可以在源代码中使用。  
  
 **错误 ID:** BC30780  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从调试代码中删除 Continue 语句。
