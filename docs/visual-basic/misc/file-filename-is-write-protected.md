# <a name="file-filename-is-write-protected"></a>文件\<文件名 > 受写入保护
尝试写入的文件受写保护，因此不能更改。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  将针对文件的权限更改为读/写，或写入另一个文件。
