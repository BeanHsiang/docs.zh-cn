# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>类型的值\<type1 > 无法转换为\<y p e 2 > 原因是数组类型的维数不同
试图将一个数组转换为另一个具有不同维数的数组。  
  
 **错误 ID:** BC30414  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改一个数组或同时更改两个数组，使两者的维数匹配。
