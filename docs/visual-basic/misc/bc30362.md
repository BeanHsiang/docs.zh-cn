# <a name="propertyname1-cannot-override-propertyname2-because-they-differ-by-readonly-or-writeonly"></a>'\<propertyname1 > 不能重写\<propertyname2 > 因为它们的差别在于 ReadOnly 还是 WriteOnly
已尝试用仅与第一个属性在 `ReadOnly` 或 `WriteOnly`上不同的第二个属性重写属性。  
  
 **错误 ID:** BC30362  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保属性不仅由 `ReadOnly` 和 `WriteOnly`区分。
