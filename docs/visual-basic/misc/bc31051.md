# <a name="namespace-or-type-name-has-already-been-imported"></a>Namespace 或类型\<名称 > 已导入
多次尝试导入命名空间。  
  
 **错误 ID:** BC31051  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `Imports` 语句。
