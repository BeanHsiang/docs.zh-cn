# <a name="this-operation-can-only-be-done-when-the-file-is-closed"></a>此操作仅可在文件关闭时完成。
当文件打开时，已尝试执行仅可在文件关闭时执行的操作。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   关闭文件，然后重试该操作。
