# <a name="operator-operatorname-is-not-defined-for-types-type1-and-type2"></a>运算符\<运算符名称 > 未定义的类型\<type1 > 和\<y p e 2 >
你已尝试使用二元运算符执行操作，但该操作对于涉及两个返回值的代码元素无效。  
  
 **错误 ID:** BC30452  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查这两个元素，并确保它们互相兼容。
