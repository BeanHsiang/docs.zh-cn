# <a name="unable-to-ping-because-a-network-connection-is-not-available"></a>由于网络连接不可用，无法使用 ping 命令
无法执行此操作，因为网络连接不可用。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   连接到网络并重新尝试该操作。
