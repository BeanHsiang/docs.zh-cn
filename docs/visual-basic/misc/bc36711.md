# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>自变量匹配参数\<parametername > 收缩到\<类型名称 >
已调用重载方法，但编译器找不到无需收缩转换便可调用的方法。 *“收缩转换”* 将值更改为可能无法保存某些可能值的数据类型。  
  
 **错误 ID:** BC36711
