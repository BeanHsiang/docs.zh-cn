# <a name="expected--at-start-of-an-embedded-expression"></a>嵌入表达式的开头应有“%=”
嵌入表达式的开始标识符 `<%=`未进行正确编码。 注意，不能在 XML 元素文本的名称中使用百分号字符 (%)。  
  
 **错误 ID:** BC31179  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保将嵌入表达式的开始标识符编码为 `<%=`。
