# <a name="me-cannot-be-the-target-of-an-assignment"></a>“Me”不能作为赋值目标
`Me` 关键字始终引用当前执行代码的类的特定实例。 其值不能更改。  
  
 **错误 ID:** BC30062  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除向 `Me`赋值的语句。
