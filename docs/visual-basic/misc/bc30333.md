# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-type3-is-not-a-reference-type"></a>类型的值\<type1 > 无法转换为\<y p e 2 > 由于 '\<type3 > 不是引用类型
语句试图将一个数组类型转换为另一个数组类型，但目标数组元素的数据类型不是引用类型。  
  
 **错误 ID:** BC30333  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查两个数组的元素数据类型以确定错误源。
