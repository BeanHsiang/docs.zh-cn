# <a name="get-is-already-declared"></a>已声明“Get”
代码块中多次出现 `Get` 语句。 `Get` 语句声明用于检索属性值的 `Get` 属性过程。  
  
 **错误 ID:** BC30443  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `Get` 语句。
