# <a name="inherits-statements-must-precede-all-declarations-in-an-interface"></a>“Inherits”语句必须位于接口中的所有声明之前
`Inherits` 语句必须是类定义中的第一个非空非注释行。  
  
 **错误 ID:** BC30357  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   移动 `Inherits` 语句，使其位于所有声明之前。
