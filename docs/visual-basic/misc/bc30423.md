# <a name="optional-parameters-cannot-be-declared-as-the-type-type"></a>可选参数不能声明为类型\<类型 >
可选参数不能具有结构的数据类型。  
  
 **错误 ID:** BC30423  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  要向可选参数传递结构，请将参数声明为 `Object`。  
  
2.  在过程中使用 `CType` 将参数强制转换为该结构类型。
