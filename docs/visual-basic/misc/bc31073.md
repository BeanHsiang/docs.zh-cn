# <a name="name1-conflicts-with-a-type-by-the-same-name-defined-in-name2"></a>'\<name1 > 与冲突\<类型 > 中定义的相同名称的\<name2 >
遇到与具有相同名称的类型冲突的名称。  
  
 **错误 ID:** BC31073  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  声明唯一的名称。  
  
2.  在冲突成员名称两端使用方括号。
