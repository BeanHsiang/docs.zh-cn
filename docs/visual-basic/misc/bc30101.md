# <a name="branching-out-of-a-finally-is-not-valid"></a>从“Finally”中分支无效
`GoTo` 块中的 `Finally` 语句在块外分支。 分支到 `Catch` 或 `Finally` 块或从两个块进行分支是无效的。  
  
 **错误 ID:** BC30101  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除 `GoTo` 语句，并考虑实现具有决策或循环控制结构的程序逻辑。
