# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>XML 特性值需要有匹配的右双引号
由双引号分隔的特性值缺少右引号字符。  
  
 **错误 ID:** BC31164  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   添加右双引号字符以分隔特性值。
