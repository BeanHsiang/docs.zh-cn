# <a name="typename-is-an-unsupported-type"></a>\<类型名称 > 是不受支持的类型
不能通过 Visual Studio 表示指定的类型。  
  
 **错误 ID:** BC30649  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用受支持的类型。  
  
-   开发新类型。
