# <a name="syntax-error-visual-basic-error"></a>语法错误（Visual Basic 错误）
源代码行生成一个非特定的语法错误。  
  
 **错误 ID:** BC30035  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查源代码行中使用的每个关键字的文档。 将文档中的示例代码与生成此错误的源行进行比较。  
  
2.  如果无法识别错误的原因，则收集有关该情况的信息并通知 Microsoft 产品支持服务。
