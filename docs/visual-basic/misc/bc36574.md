# <a name="anonymous-type-must-contain-at-least-one-member"></a>匿名类型必须至少包含一个成员
匿名类型声明中的初始值设定项列表不能为空。  
  
```  
' Not valid.  
' Dim anonInstance = New With {}  
```  
  
 **错误 ID:** BC36574  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在大括号内声明成员，如下面的代码中所示。  
  
    ```  
    Dim anonInstance = New With {.MemberName = "value"}  
    ```
