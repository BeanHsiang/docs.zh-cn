# <a name="no-default-member-found-for-type-typename"></a>没有默认成员找到类型\<类型名称 >
已尝试后期绑定索引 `Get` 或 `Set` 调用。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  确保该对象具有带 `Get` 或 `Set`的默认属性，具体取决于你试图进行的操作。
