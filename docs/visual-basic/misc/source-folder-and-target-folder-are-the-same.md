# <a name="source-folder-and-target-folder-are-the-same"></a>源文件夹和目标文件夹是相同的
尝试复制或移动文件，但指定的源目录和目标目录是相同的。 这可能导致不正确地分析文件路径。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保提供正确的源和目标目录。
