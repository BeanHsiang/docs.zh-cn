# <a name="membername-has-the-same-name-as-a-type-parameter"></a>\<成员名称 > 与类型参数同名
泛型类或结构的一个成员与该类或结构的一个类型参数同名。  
  
 每个变量、常量、枚举、过程、属性、事件和委托必须具有与每个类型参数不同的名称。  
  
 **错误 ID:** BC32054  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改类或结构成员的名称，以使它与每个类型参数的名称都不同。
