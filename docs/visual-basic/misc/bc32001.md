# <a name="keyword-is-not-valid-within-a-module"></a>\<关键字 > 在模块中无效
模块内使用与类实例相关的关键字，如 `Me` 或 `MyBase`。 模块没有继承或实例。  
  
 **错误 ID:** BC32001  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   如果使用关键字的代码涉及类实例，则将其移到类实现。  
  
-   如果使用关键字的代码适用于该模块，则删除无效的关键字。
