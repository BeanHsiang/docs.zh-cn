# <a name="overload-resolution-failed-because-no-method-is-accessible"></a>重载决策失败，因为没有\<方法 > 可访问
调用了一个重载方法，但编译器找不到参数列表在参数个数上匹配的可访问方法。  
  
 **错误 ID:** BC30517  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查所提供的参数列表。  
  
2.  检查试图调用的方法，以确定其是否可访问。
