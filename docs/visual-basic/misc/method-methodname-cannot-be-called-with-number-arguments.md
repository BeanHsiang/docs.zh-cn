# <a name="method-methodname-cannot-be-called-with-number-arguments"></a>方法\<m h o d > 不能使用调用\<数 > 自变量
方法调用包含错误的参数数量。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  对你已提供的参数进行计数。  
  
2.  将数量与你想调用的方法中的参数的数量进行匹配。  
  
3.  如果两个数量不一致，请调整参数的数量。
