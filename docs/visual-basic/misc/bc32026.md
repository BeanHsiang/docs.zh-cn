# <a name="specifiers-and-attributes-are-not-valid-on-namespace-statements"></a>说明符和特性在“命名空间”语句上无效
命名空间用于程序块和编程元素的分层组织。 它们不采用任何修饰符或特性。  
  
 **错误 ID:** BC32026  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从 `Namespace` 语句删除所有关键字和特性块。
