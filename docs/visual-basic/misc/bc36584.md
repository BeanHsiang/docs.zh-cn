# <a name="parameter-parametername-in-extension-method-methodname-defined-in-typename-already-has-a-matching-argument"></a>参数\<parametername > 中的扩展方法\<m h o d > 中定义\<类型名称 > 已具有匹配的实参
参数在同一过程调用中按名称传递了多次。  
  
 **错误 ID:** BC36584  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查以确定是否其中一个参数名拼写错误。  
  
-   从参数列表中删除重复的参数。
