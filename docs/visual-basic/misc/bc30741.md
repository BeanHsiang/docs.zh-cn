# <a name="typename1-cannot-be-converted-to-typename2"></a>'\<typename1 > 无法转换为\<typename2 >
在两个不兼容的类型之间尝试执行了类型转换。  
  
 **错误 ID:** BC30741  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   验证正在被强制转换的类型。
