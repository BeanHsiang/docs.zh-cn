# <a name="error-embedding-win32-manifest-manifest"></a>嵌入 Win32 清单时出错：\<清单 >
无法嵌入为 `/win32manifest` 编译器选项指定的文件。 这可能是对清单文件的安全性特权不足造成的。  
  
 **错误 ID:** BC31191  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  确保运行 Visual Basic 编译器命令的标识有权访问 Win32 清单文件。
