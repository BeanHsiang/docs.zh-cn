# <a name="line-number-cannot-be-parsed-using-the-current-fieldwidths"></a>行\<数 > 无法使用当前的 FieldWidths 分析
无法分析指定的行，因为其字段的宽度与所指定的不同。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   调整 `FieldWidths` 以便正确分析此行，或插入异常处理代码以处理此行。
