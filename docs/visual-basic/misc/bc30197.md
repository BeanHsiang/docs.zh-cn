# <a name="as-expected"></a>应为“As”
声明出现，它不会为一个或多个已声明的元素提供 `As` 子句。  
  
 **错误 ID:** BC30197  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   按照要求提供 `As` 子句，或返回 `Option Strict Off`。
