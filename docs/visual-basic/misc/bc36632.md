# <a name="nullable-parameters-must-specify-a-type"></a>可为 null 的参数必须指定类型
参数包含可为 null 的修饰符（?），但未指定类型。  
  
 **错误 ID:** BC36632  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   声明可为 null 的参数时，请指定一种类型。
