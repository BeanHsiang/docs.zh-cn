# <a name="method-call-did-not-return-a-value"></a>方法调用未返回值
调用的方法未按预期返回值。  
  
 **错误 ID:** BC30723  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   验证调用的方法是否为会返回值的函数。
