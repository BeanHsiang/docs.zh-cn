# <a name="inherits-not-valid-in-modules"></a>模块中的“Inherits”无效
`Inherits` 语句发生在模块内。 模块不能从类继承。  
  
 **错误 ID:** BC30230  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除 `Inherits` 语句或重新输入作为类的模块。
