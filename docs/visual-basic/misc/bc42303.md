# <a name="xml-comment-cannot-appear-within-a-method-or-a-property"></a>XML 注释不能在方法或属性内出现
XML 注释不能在方法或属性内出现。 将忽略该 XML 注释。 XML 注释块不能出现在方法或属性内部。  
  
 **错误 ID:** BC42303  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除注释块，或将其移动到更适合的位置。
