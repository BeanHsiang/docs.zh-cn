# <a name="reference-required-to-assembly-assemblyname-containing-the-type-membername"></a>需要对程序集的引用\<程序集名称 > 包含类型\<成员名称 >
需要对程序集的引用\<程序集名称 > 包含类型\<成员名称 >。 请向项目中添加一个。  
  
 项目包含一个类型，该类型依赖于项目引用中未包括的程序集中的另一个类型。  
  
 **错误 ID:** BC30652  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在项目中添加依赖类型的项目引用。
