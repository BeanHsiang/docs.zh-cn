# <a name="declare-statements-in-a-structure-cannot-be-declared-specifier"></a>Declare 语句在结构中的不能声明为\<说明符 >
`Declare` 声明具有在 `Structure` 声明中无效的说明符。  
  
 **错误 ID:** BC30791  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除说明符。
