# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>方法声明语句必须位于逻辑行的开头。
一个 `Function` 或 `Sub` 语句跟在冒号 (:) 语句分隔符后面。 `Function` 或 `Sub` 必须是其源行的唯一语句。  
  
 **错误 ID:** BC32009  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将多个语句分隔到不同的行。
