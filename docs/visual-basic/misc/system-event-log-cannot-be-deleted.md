# <a name="system-event-log-cannot-be-deleted"></a>不能删除系统事件日志
试图删除系统事件日志（该日志是不能被删除的）。 系统日志跟踪系统事件（如系统启动和硬件故障）。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请考虑让应用程序写入应用程序或自定义日志，而不是系统事件日志。  
  
-   请勿试图删除系统事件日志。
