# <a name="addhandler-is-already-declared"></a>已经声明 "AddHandler"
自定义事件声明中出现多个 `AddHandler` 声明。 `AddHandler` 声明会声明用于添加事件处理程序的过程。  
  
 **错误 ID:** BC31127  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `AddHandler` 语句。
